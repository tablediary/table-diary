# 桌面日记与备忘录二合一应用规格说明

## 1. 项目概述
- **项目名称**: 书桌日记
- **类型**: 单页面Web应用
- **核心功能**: 整合随笔日记与备忘录的桌面端应用，支持本地存储和定时提醒
- **目标用户**: 需要在电脑上记录日常想法和待办事项的用户

## 2. UI/UX 规格

### 布局结构
- **整体**: 全屏木质书桌背景
- **书页区域**: 居中显示的奶白色书页（随笔展示区）
- **便利贴区域**: 书页上方，淡黄色便签列表
- **顶部导航**: 类型切换toggle + 日期选择器
- **底部**: 输入区域

### 视觉设计

#### 色彩方案
- **主背景**: #8B7355 (浅棕色木桌)
- **木纹纹理**: 使用CSS渐变模拟
- **书页**: #FFFEF5 (奶白色/象牙白)
- **便利贴**: #FFF9C4 (淡黄色)
- **卡片背景**: #FAF8F0 (米白色，比书页浅)
- **主色调**: #8B5A2B (暖棕色)
- **主色调Hover**: #6B4423 (深暖棕)
- **文字主色**: #4A3728 (深棕)
- **文字次色**: #9E9E9E (浅灰色)
- **边框**: #D4C4B0 (浅棕)

#### 字体
- **主字体**: "Noto Serif SC", "STSong", serif (衬线体营造书本感)
- **图标字体**: 无需额外加载，使用SVG内联

#### 间距
- **书页内边距**: 32px
- **卡片内边距**: 16px
- **卡片圆角**: 8px
- **元素间距**: 16px

### 组件

#### 1. 类型切换Toggle
- 精致toggle样式，暖棕色高亮选中
- 选项：随笔 / 备忘录
- 选中态背景：#8B5A2B
- 未选中背景：#D4C4B0

#### 2. 日期选择器
- 简洁日历样式
- 选中日期高亮

#### 3. 随笔卡片
- 米白色背景
- 圆角8px
- 左侧竖线装饰（暖棕色）
- 显示内容、时间戳
- 右侧铅笔图标(编辑) + 垃圾桶图标(删除)
- hover显示文字提示

#### 4. 备忘录便利贴
- 淡黄色背景
- 细阴影营造浮起效果
- 事项内容、开始时间、提醒时间
- 同样支持编辑/删除

#### 5. 输入区域
- 暖棕系边框
- textarea/输入框
- 提交按钮（暖棕色+hover变色）

#### 6. 提醒弹窗
- 居中模态框
- 暖棕主题
- 包含提醒内容和关闭按钮

## 3. 功能规格

### 核心功能

#### 3.1 记录类型
- **随笔**: 纯文字内容，按日期查看
- **备忘录**: 包含事项内容、开始时间、提醒时间

#### 3.2 添加记录
- 点击添加按钮弹出输入框
- 随笔：只需输入文字内容
- 备忘录：需填写内容、开始时间、提醒时间
- 同一日期可无限添加多条

#### 3.3 编辑记录
- 点击铅笔图标进入编辑模式
- 修改后保存

#### 3.4 删除记录
- 点击垃圾桶图标
- 确认后删除

#### 3.5 日期切换
- 选择不同日期查看该日记录
- 默认显示今日

#### 3.6 本地存储
- 使用localStorage存储
- 结构：{ date: { essays: [], memos: [] } }

#### 3.7 定时提醒
- 检查备忘录提醒时间
- 到达时自动弹窗提醒
- 每分钟检查一次
